
## Installation

``` bash
$ npm install insertion-sort-v --save
```

## Usage

``` javascript
var sort = require('insertion-sort'),
    example_array = [4,1,3,2],
    sorted_array;

sorted_array = sort.asc(example_array);

console.log(sorted_array);
```
